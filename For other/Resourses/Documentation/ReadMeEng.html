Windows 10 Tweak






Tweak made by Michael Maliar.
GitHub: https://github.com/MichaelMaliar/Windows-10-Tweak.git


The tweak was created to quickly configure Windows. It is designed to be used on versions of Windows 1903 or 1809 (but a few settings will not be changed).
You are solely responsible for using the software on stripped-down Windows builds or older versions of the operating system.
If you have any questions — write on e-mail.

The tweak is adapted for Windows in Russian, Ukrainian, Belarusian, Kazakh and Moldovan. On Windows with another language tweak will be in English.
Before you start, you must disable the "Tamper Protection" function in Windows Defender. It is on the way: «Settings\Update & Security\Windows Security\Virus & threat protection\Virus & threat protection settings\Manage settings\Tamper Protection: off»
It can be run by double-clicking on the exe file and from the folder where it was unpacked by clicking on the Setup file.cmd, but all files should be next to it in the same directory!

If you are going to change something, follow the instructions:

1) SFX comments:
Path=%SystemRoot%\Windows_10_Tweak — Unpack to Windows\Windows_10_Tweak folder.
SavePath — Allows you to change the suggested path to the desired one only once, rather than re-enter it each time.
Silent=1 — Quiet unpacking.
Overwrite=1 — Overwrite files if they already exist.
Setup=Setup.cmd — Running the tweak.
Shortcut=D, "%SystemRoot%\System32\wscript.exe //B //Nologo %systemroot%\Refresh.vbs", , "Restart Explorer", "Refresh", Refresh.ico — Creates a shortcut to reboot the Explorer shell.

The TamperProtection.vbs and done.vbs files are used for notifications. Utility SubInACL.exe need to manage permissions, Utility LGPO.exe needed to manage Local group policies (gpedit.msc в Windows 10 Pro). The GPO folder contains group policies. File Blank.ico is used to remove the arrows from the labels, Refresh.ico is a Refresh shortcut icon on the desktop. Delete_Intel_folder.xml contains a task to delete the Intel folder for the task scheduler. Settings.txt contains recommendations for manual configuration. PowerRun.exe is needed to run processes with elevated privileges. Script Refresh.vbs silently updates the Explorer and resets the icon cache.
