Windows 10 Tweak






Твик сделал Маляр Михаил.
GitHub: https://github.com/MichaelMaliar/Windows-10-Tweak.git


Tвик был создан для быстрой настройки Windows. Он расчитан на использование на версиях Windows 1903 или 1809 (но несколько параметров не будут изменены).
За использование программы на урезанных сборках Windows или старых версиях операционной системы ответственность несёте только вы.
Если будут какие-то вопросы — пишите на почту.

Tвик адаптирован для Windows на русском, украинском, белорусском, казахском и молдавском языках. На Windows с другим языком твик будет на английском.
Перед запуском необходимо отключить функцию "Защита от подделки" в Windows Defender. Она находится по пути: «Параметры\Обновление и безопасность\Безопасность Windows\Защита от вирусов и угроз\Параметры защиты от вирусов и других угроз\Управление настройками\Защита от подделки: откл»
Его можно запускать двойным кликом по exe файлу так и из папки, куда он был распакован, кликом по файлу Setup.cmd, но все файлы должны лежать рядом с ним в одной директории!

Eсли вы собираетесь что-то менять, придерживаетесь инструкции:

1) SFX-комментарии:
Path=%SystemRoot%\Windows_10_Tweak — Распаковка в папку Windows\Windows_10_Tweak.
SavePath — Позволяет изменить предложенный путь на желаемый лишь единожды, а не вводить его повторно каждый раз.
Silent=1 — Тихая распаковка.
Overwrite=1 — Перезапись файлов если они уже существуют.
Setup=Setup.cmd — Запуск твика.
Shortcut=D, "%SystemRoot%\System32\wscript.exe //B //Nologo %systemroot%\Refresh.vbs", , "Restart Explorer", "Refresh", Refresh.ico — Создает ярлык для перезагрузки оболочки проводника.

2) SFX-модуль:
ВАЖНО! При создании SFX-архива нужно выбрать 64-битный SFX модуль — Default64.SFX.

3) Как создать SFX-архив:


Файлы TamperProtection.vbs, done.vbs, служат для оповещения. Утилита SubInACL.exe нужна для управления разрешениями, утилита LGPO.exe нужна для управления Локальными групповыми политиками (gpedit.msc в Windows 10 Pro). Папка GPO содержит групповые политики. Файл Blank.ico служит для того чтобы убрать стрелки с ярлыков, Refresh.ico иконкой ярлыка Refresh на рабочем столе. Delete_Intel_folder.xml содержит в себе задание по удалению папки Intel для планировщика заданий. Settings.txt содержит в себе рекомендации для ручной настройки. PowerRun.exe нужна для запуска процессов с повышенными привелегиями. Скрипт Refresh.vbs тихо обновляет проводник и сбрасывает кэш иконок.
